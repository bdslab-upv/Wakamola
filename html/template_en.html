Wakamola Network Visualiser

In the image you can see the Wakamola Social Network, each circle is a Wakamola user and the connections (gray lines) are the relationships that are created when sharing Wakamola. The circles show the value of the Body Mass Index (BMI) of the users, the colors indicate if it is a BMI within the acceptable limits (green), overweight (yellow), with obesity (red) or with less weight than acceptable (blue). You can change the type of display with the slider bar (IMC / Wakastatus) to see the Wakaestado values ​​(scale from 0 to 100) of the users, in this case the colors show the level of obesity risk from green (without risk) to red (with risk). If you click on communities button you will see the user groups that are created in the social network and that may be influencing each other.
